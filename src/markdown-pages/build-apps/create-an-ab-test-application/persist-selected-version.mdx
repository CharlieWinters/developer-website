In the previous lesson, you created a section of your application that allows you to pick the most effective design from your A/B test. The goal of this section is to end the test once you’ve selected a winner. Here’s what you have so far:

![First version of your end test section](../../../images/ab-test/end-test-section-v1.png)

When you select a version from this form, the selection does not persist in the `Select` component. Now, it’s time to implement that behavior.

<Steps>

<Step>

Change to the `persist-selected-version` directory of the course repository:

```sh
cd nru-programmability-course/persist-selected-version
```

</Step>

<Step>

Open your Nerdlet's _index.js_ file. All code you write in this lesson belongs in this file.

</Step>
