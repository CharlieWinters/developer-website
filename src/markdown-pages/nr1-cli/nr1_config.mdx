<Intro>

To configure your New Relic One CLI preferences, use the commands below. You can click any command to see their specific details, including usage and options.

</Intro>

| Command | Description |
|---|---|
| [`nr1 config:set`](#set) | Sets a specific configuration value. |
| [`nr1 config:get`](#get) | Shows a specific configuration. |
| [`nr1 config:list`](#list) | Lists your configuration choices. |
| [`nr1 config:delete`](#delete) | Removes the value of a specific configuration. |

***

## `nr1 config:set` {#set}
### Set a configuration
Sets a specific configuration value. *(Maybe some additional color?)*

### Usage
`$ nr1 config:set OPTION` 

### Options
| Option | Description |
|---|---|
| `-k, --key=KEY` | The key of the config. **(required)** |
| `-V, --value=VALUE` | The value of the config. |
| `--profile=PROFILE` | The authentication profile you want to use. |
|` -t, --this-profile-only` | If present, this configuration will only apply while running with the specified profile.|
| `--verbose` | Adds extra information to the output. |

## `nr1 config:get` {#get}
### See your configuration
Shows the value for a specific configuration.

### Usage
`$ nr1 config:get OPTION`

### Options
| Option | Description |
|---|---|
| `-k, --key=KEY` | The key of the config. **(required)** |
| `--profile=PROFILE` | The authentication profile you want to use. |
| `--verbose` | Adds extra information to the output. |

## `nr1 config:list` {#list}
### See all your configurations
Shows a list of all your configuration choices.

### Usage
`$ nr1 config:list`

### Options
| Option | Description |
|---|---|
| `--profile=PROFILE` | The authentication profile you want to use. |
| `--verbose` | Adds extra information to the output. |

### Aliases
* `$ nr1 config:ls`
